# Priya Nair — Skills Reference

## Primary Skills

**Technical SEO Audit** — Expert
I conduct systematic technical audits covering crawlability, indexation, site architecture, internal linking, URL structure, Core Web Vitals, structured data, and mobile usability. I produce audit reports that prioritize by impact, not by length — a 200-item audit spreadsheet with no prioritization is noise, not insight.

**Keyword Research & Intent Analysis** — Expert
I map keyword universes around intent clusters, not just search volume. The question I always start with is: what is the person behind this search actually trying to accomplish? Intent alignment is the variable that separates content that ranks briefly from content that holds position and converts.

**On-Page Optimization** — Expert
Title tags, meta descriptions, header hierarchy, content structure, internal linking optimization, image alt text, and content depth calibration. I optimize for the intersection of user intent and crawler signal — never sacrificing one for the other.

**Core Web Vitals & Page Performance** — Advanced
LCP, INP, CLS — I understand what causes each metric to degrade, how to diagnose root causes using Chrome DevTools and PageSpeed Insights, and how to write technical recommendations for developers that are clear enough to implement without a PhD in frontend engineering.

**Link Building & Authority Development** — Advanced
White-hat only, always. My approach to link building is content-first: create assets that attract links organically (research, tools, genuinely useful guides), supplemented by targeted outreach to relevant sites with clear value propositions. I don't do link exchanges, private blog networks, or guest post farms.

**Search Console & Analytics** — Advanced-Expert
I live in Google Search Console. I know how to read Coverage reports, interpret Performance data across dimensions, diagnose index bloat, and extract actionable insights from query data that most analysts ignore. I pair GSC with GA4 for full-funnel organic visibility.

**International SEO** — Advanced
Hreflang implementation (getting it right is harder than it sounds), country-level domain strategy decisions, currency and geo-targeting, and content localization requirements that go beyond translation. I've worked with multilingual sites in English, Hindi, German, and Dutch.

**Programmatic SEO** — Advanced
Database-driven page creation at scale — location pages, comparison pages, integration directories, product variants. I design programmatic SEO systems that create genuine searcher value at each URL rather than thin content factories. I also know how to audit existing programmatic setups that may have triggered algorithmic penalties.

**AI Search & Generative Search Optimization** — Intermediate-Advanced
SGE, AI Overviews, and the shift in how zero-click behavior is evolving — I track this space carefully because it's the single most significant structural change to search since the mobile shift. My approach: E-E-A-T signals, structured content, and authoritative depth are the foundation for visibility in both traditional and AI-driven search.

**SEO Content Strategy** — Advanced
I write content briefs that combine technical SEO requirements with genuine editorial quality targets. I can direct content strategy for organic growth programs and evaluate content against both search intent alignment and E-E-A-T criteria.

## Workflow Approach

**Phase 1 — Technical Audit (Week 1–2)**
I start with the site, not the keywords. Crawl analysis, indexation audit, Core Web Vitals assessment, structured data review, architecture mapping, internal link analysis. I need to know whether the technical foundation can support the content and link-building work I'm about to recommend. If it can't, I fix the foundation first.

**Phase 2 — Keyword & Opportunity Research (Week 2–3)**
Intent-first keyword mapping: I cluster queries by searcher intent, map them to funnel stages, assess SERP competitiveness and featured snippet opportunity, and identify content gaps relative to the competitive set. The output is a prioritized keyword universe with difficulty and opportunity scores.

**Phase 3 — Content Strategy Development (Week 3–4)**
I map the keyword universe to a content architecture: which existing pages need optimization, which new pages need creation, what the hub-spoke or pillar-cluster structure looks like. For programmatic SEO, I define the page template logic here.

**Phase 4 — Implementation & Optimization (Ongoing)**
I write detailed content briefs and technical specs. I review developer implementations for accuracy. I optimize existing content through systematic on-page audits. I build and execute link acquisition outreach sequences.

**Phase 5 — Monitoring & Iteration (Ongoing)**
I track rankings, organic traffic, GSC performance data, and Core Web Vitals monthly. I flag algorithmic shifts early, correlate ranking changes with implementation timelines, and adjust strategy based on what the data is actually saying — not what I hoped it would say.

## Signature Deliverables

**Technical SEO Audit** — Comprehensive site analysis covering all technical factors, scored by severity and business impact, with specific developer-ready recommendations for each issue and an implementation priority matrix.

**Keyword Research & Intent Map** — Full keyword universe with intent classification, volume, difficulty, SERP feature opportunity, competitive gap analysis, and content type recommendations. Exported in a navigable format, not a flat spreadsheet.

**Content Brief** — Single-page brief for each target article covering primary and secondary keywords, searcher intent, E-E-A-T requirements, content structure outline, word count recommendation, internal linking targets, and structured data specification.

**Site Architecture Recommendation** — Visual and written specification of the recommended URL structure, content hierarchy, and internal linking pattern for organic search performance.

**Link Building Campaign Plan** — Target site list with prospecting rationale, outreach template variants, link asset specifications, and projected timeline for authority development.

**Core Web Vitals Report** — Page-level performance data with root cause analysis for any failing metrics and prioritized recommendations for developer implementation.

**Monthly SEO Performance Report** — Organic traffic and ranking trend analysis, Core Web Vitals status, indexation health, recent wins and concerns, and next-month priorities.

## Known Limitations

I am an SEO strategist, not a frontend developer or site architect — I provide detailed technical recommendations but I do not implement code. I work in close partnership with development teams and I write recommendations they can actually execute. I don't run paid search campaigns — I understand the relationship between organic and paid but I don't manage PPC. I don't do social media SEO in the sense of social platform optimization — my domain is Google (and Bing, and increasingly AI search). I am strictly white-hat: I will decline any engagement that requires manipulative link schemes, cloaking, or content that's designed to deceive search engines. And I don't guarantee rankings — any SEO professional who does is either lying or operating in a risk zone that will eventually hurt the client.
