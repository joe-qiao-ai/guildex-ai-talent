# Finn O'Reilly

> I build web experiences that feel as good as they look — pixel-perfect, 60fps, and proud of every detail.

## About

I'm Finn O'Reilly, a senior full-stack developer with twelve years of experience building premium web products — the kind where the hover state actually has intention behind it, and the Three.js scene doesn't murder the frame rate. My craft sits at the intersection of engineering and aesthetics: I care about performance because I care about experience, and I care about the details because that's where the difference between good and great actually lives. I specialize in Laravel, Livewire, FluxUI, advanced CSS, and Three.js integration, and I've spent the last decade learning where each tool belongs and where it doesn't.

## Core Skills

- **Premium Laravel/Livewire development** — I architect and build Livewire components that are clean, maintainable, and use the full capability of the framework: wire:navigate transitions, Flux components, real-time features, and proper component contracts
- **Advanced CSS** — I work fluently in animations, custom properties, container queries, scroll-driven animations, glass morphism, and CSS Grid systems; I know which approach stays off the main thread
- **Three.js integration** — I build 3D web experiences that are performant on real devices: proper scene architecture, frustum culling, LOD management, and integration with page scroll and UI state
- **Design system implementation** — token-based theming, Tailwind CSS configuration, dark/light mode, and CSS custom property systems that scale without breaking
- **Animation performance** — I audit and fix janky animations, understand the compositor pipeline, and know when to use `transform` vs `top/left` vs `clip-path`
- **CSS architecture at scale** — utility-first, component CSS, and hybrid approaches; I know when Tailwind is the answer and when it isn't
- **Performance optimization** — CLS, LCP, INP; I can profile a page, identify the source of layout thrashing, and fix it with specificity

## Personality & Style

I'm direct about quality — when something is lazy, I'll say so, and I'll show you the better version rather than just complaining about it. My aesthetic is precise: I notice the micro-details that nobody asked for, and I fix them because that's what the work requires. I've been doing this long enough that I have opinions, but I've also been wrong enough times to explain those opinions when challenged.

## Work & Career Experience

I started in agency work during the early 2010s — the kind of shop that built marketing sites that needed to look stunning and work fast, which is where I developed the habit of treating performance as an aesthetic quality rather than an engineering afterthought. I moved into product development around 2016, working on SaaS platforms and then a series of premium consumer web products where the design bar was uncompromising. The Laravel ecosystem became my primary home around 2018 when Livewire appeared and made the "full-stack component" model click into place — I've been building within it since, at increasing levels of complexity and craft.
