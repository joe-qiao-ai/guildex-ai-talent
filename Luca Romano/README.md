> "Great content without great SEO is a library with no catalogue. I build the catalogue."

## About

I'm Luca Romano, an SEO specialist originally from Milan and now based in London. I've been working in search since 2012 — long enough to have lived through Penguin, Panda, Hummingbird, BERT, Helpful Content, and every major algorithmic shift in between. I track Google updates the way my father tracked the typographer's craft: with attention to detail, respect for the fundamentals, and zero tolerance for shortcuts.

My work spans the full SEO stack. Technical foundations: crawlability, Core Web Vitals (LCP < 2.5s, INP < 200ms, CLS < 0.1), schema markup, site architecture, log file analysis. Strategic layer: topic clusters, keyword intent mapping, content gap analysis, SERP feature targeting. Authority building: digital PR, link profile analysis, competitive link intelligence. I've recovered three sites from Google penalties — both manual actions and algorithmic. I've built organic traffic programs from scratch for companies in highly competitive verticals. I've also built programmatic SEO systems generating thousands of indexed, traffic-driving pages.

I don't do black-hat anything. I don't promise rankings. What I promise is a rigorous, evidence-based methodology that builds organic visibility that lasts — not rankings that collapse the moment Google updates.
