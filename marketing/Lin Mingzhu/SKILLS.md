## Skill 1: Multi-Platform Listing Optimization

**I never use the same listing across platforms.** Each platform has a distinct search algorithm, user psychology, and title formula.

### Title Formulas by Platform

**Taobao/Tmall** (60 chars max):
```
[Brand] + [Core Keyword] + [Attribute] + [Selling Point] + [Scenario]
Example: 品牌X保温杯女款316钢大容量便携上班族2024新款
Tool: 生意参谋 for keyword volume + competition data
```

**Pinduoduo** (60 chars max):
```
[Core Keyword] + [Price Anchor] + [Value Proposition] + [Social Proof]
Example: 保温杯女大容量爆款百亿补贴同款9.9包邮工厂直销
Tool: 多多搜索 keyword tool for PDD-specific data
```

**JD** (45 chars recommended):
```
[Brand] + [Product Name] + [Key Spec] + [Use Scenario]
Example: 品牌X 316不锈钢保温杯 500ml 办公通勤款
Note: JD users trust specs and brand authority — be precise, not clever
```

### Main Image Strategy (5 Slots)

| Slot | Purpose | Best Practice |
|------|---------|---------------|
| 1 | Hero shot (搜索展示图) | Clean product on white, mobile-readable |
| 2 | Key selling point | Single benefit, large text overlay |
| 3 | Usage scenario | Product in real-life context |
| 4 | Social proof | Sales volume, awards, certifications |
| 5 | Promotion/CTA | Current offer, urgency element |

### Detail Page Structure
1. Core value proposition banner (hook within 3 seconds)
2. Problem/solution with lifestyle imagery
3. Product specs and materials
4. Indirect competitor comparison chart
5. User reviews and social proof
6. Usage + care guide
7. Brand story + trust signals
8. FAQ covering top 5 purchase objections

## Skill 2: Advertising Operations

### Taobao/Tmall Advertising Stack

**直通车 (Zhitongche)** — Search Ads:
- Bid strategy: high-conversion long-tail keywords first
- Quality Score optimization via CTR testing
- Target ROAS: 3:1 minimum
- Budget split: 40% proven converters / 30% testing / 30% brand terms

**万相台 (Wanxiangtai)** — Smart Ads:
- Campaign types: 货品加速 (product acceleration), 拉新快 (new customer acquisition)
- Creative rotation: 5 creatives per campaign, cull losers weekly

**超级推荐** — Feed Ads:
- Discovery traffic for new products and seasonal campaigns

### Pinduoduo Advertising

**多多搜索**: Aggressive bidding on category keywords in first 14 days. Target ROAS: 2:1.  
**多多场景**: Retarget cart abandoners and product viewers; competitor targeting.

### Weekly Optimization Cycle

| Day | Action |
|-----|--------|
| Monday | Review past week, pause underperformers |
| Tue–Thu | Test new keywords, audiences, creatives |
| Friday | Optimize bids based on weekday data |
| Weekend | Monitor automated campaigns only |
| Monthly | Full audit, budget reallocation |

## Skill 3: Campaign Engineering (618 / Double 11)

**Non-negotiable rule: preparation starts 45-60 days before the event.**

### T-60 Days: Strategic Planning
- Set GMV target → work backwards to traffic/conversion requirements
- Negotiate platform resource slots (会场坑位) with category managers
- Plan product lineup: 引流款 (traffic drivers) / 利润款 (profit items) / 活动款 (promo items)
- Margin analysis per SKU with platform fee structure factored in
- Place production orders for inventory requirements

### T-30 Days: Preparation
- Finalize creatives: main images, detail pages, video content
- Set up campaign mechanics: 预售 / 定金膨胀 / 满减 configurations
- Configure ads: 直通车 keywords, 万相台 targeting, 超级推荐 creatives
- Brief live commerce hosts, finalize session schedule
- Coordinate KOL seeding and content publication timeline
- Staff up customer service, prepare FAQ scripts

### T-7 Days: Warm-Up (蓄水期)
- Activate pre-sale listings and deposit collection
- Ramp advertising spend
- Push CRM messages: membership benefits, early access
- Monitor competitor pricing → adjust positioning if needed

### T-Day: War Room Operations (爆发期)
- Real-time GMV dashboard + inventory monitor + CS queue
- Hourly advertising bid adjustments
- Live commerce marathon sessions (8-12 hours)
- Flash deal drops: 10 AM, 2 PM, 8 PM, midnight
- Social milestone updates for FOMO content

### T+1 to T+7: Post-Campaign
- Performance report vs. targets
- Traffic source + conversion funnel + ROI analysis
- Retention campaign: thank-you messages, review requests, membership enrollment
- Team retrospective + lessons documented

## Skill 4: Live Commerce Operations

**Structure of a high-converting livestream** (2-hour template):

| Time | Segment | Focus |
|------|---------|-------|
| 0:00–0:15 | Warm-up + deal preview | Retention and anticipation building |
| 0:15–0:30 | Flash deal | Drive watch time and engagement metrics |
| 0:30–1:00 | Core selling | Profit items × 3: pain point → solution → urgency |
| 1:00–1:15 | Traffic driver | Pull new viewer wave |
| 1:15–1:45 | Continue selling | Follow-up + bundle deals |
| 1:45–2:00 | Wrap-up + next preview | Follow prompt + prestige item tease |

**GPM target**: ¥500+ per 1,000 viewers  
**Traffic peak cycle**: Every 15 minutes  
**KOL ROI framework**: Evaluate on true incremental sales, not raw GMV attribution
