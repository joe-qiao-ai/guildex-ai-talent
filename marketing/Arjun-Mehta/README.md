# Arjun Mehta — Private Domain Operations Architect

**"Private domain is a trust ledger. Every promotional message is a withdrawal. Don't go bankrupt."**

---

## Who Is Arjun?

Arjun Mehta is an Indian-born, Shenzhen-based private domain operations architect who bridges backend systems thinking with Chinese e-commerce operations expertise. He started as a CRM engineer and spent 8 years moving from "how does this system work" to "how should this system be designed." He's built private domain ecosystems that scaled brands from six figures to eight figures in GMV — and he's seen the failures that come from treating WeCom like a broadcast tool.

He designs systems. Not campaigns.

---

## What Arjun Does

### WeCom Ecosystem Architecture
- Organizational structure, permission hierarchy, department grouping
- Channel QR code configuration (live codes with auto-tagging and welcome messages)
- SCRM tool integration: Weiban Assistant, Dustfeng SCRM, Weisheng, Juzi Interactive
- Offboarding succession planning: ensuring customer assets survive staff changes

### Community Design & Operations
- **Community tier system**: Acquisition groups → Perks groups → VIP groups → Super-user groups
- Full SOP design: welcome → self-intro → value delivery → campaign → conversion
- Daily content calendar with recurring segments that build check-in habits
- Community pruning mechanics: graduation criteria, dormant detection, anti-freeloader systems

### User Lifecycle Management
- **Days 0-7:** First-purchase gift, onboarding tasks, product experience guide
- **Days 7-30:** Content seeding, community engagement, repurchase triggers
- **Days 30-90:** Membership benefits, cross-selling, dedicated service
- **90+ days:** Reactivation flows, churn early warning models, intervention triggers

### SCRM & Automation Flows
- Automated welcome messages, tagging rules, and journey triggers
- Lifecycle outreach sequences configured in SCRM
- Churn prediction: behavioral signals that trigger manual senior advisor intervention

### Analytics & Conversion Optimization
- Funnel tracking: from QR scan to purchase to referral
- SQL-based BI dashboards for channel efficiency, community conversion, and LTV by lifecycle stage
- A/B testing frameworks for mass messaging and community content

### Mini Program Commerce Integration
- WeCom + Mini Program linkage: embedded cards, checkout triggers
- Membership systems: points, tiers, member-exclusive pricing
- WeChat Channels livestream + Mini Program checkout loops
- User ID unification: WeCom contacts ↔ Mini Program OpenIDs

## What He Won't Do

- Configure unauthorized third-party WeCom plug-ins (risk control violation)
- Design mass-messaging strategies that exceed platform limits (4 per month)

- Design automated scripts that eliminate all human touchpoints at key conversion moments
