## Skill 1: Hook Engineering (First 3 Seconds)

**The first 3 seconds determine whether the algorithm shows your video to 1,000 or 100,000 people.**

### Four Hook Types (in order of typical performance)

**A. Conflict Hook**
```
"Never buy [product category] until you watch this first."
"Stop using [common product] — here's what it's doing to you."
```
*Best for*: Product seeding, health/lifestyle categories

**B. Value Hook**
```
"Spent ¥[X] to find the best [category] — here's what I learned."
"This one thing saved me [3 hours / ¥500 / 2 months] a [week/month]."
```
*Best for*: Educational content, product reviews

**C. Suspense Hook**
```
"I discovered something the [industry] doesn't want you to know."
"Watch what happens at second 25 — I wasn't expecting this."
```
*Best for*: Drama/narrative formats, unboxing

**D. Relatability Hook**
```
"Does anyone else [extremely specific frustration]?"
"POV: You've been doing [common thing] wrong for years."
```
*Best for*: Lifestyle, daily-life content, younger audiences

### Hook Testing Protocol
- Write 3 different hooks for every video idea
- A/B test with DOU+ targeted pushes (¥50-100 per variant)
- Track completion rate at 3s, 10s, 50% mark
- Archive winners in a running hook library

## Skill 2: Video Script Structure

### Full Script Template

```markdown
## Video Script

**Target duration**: 30-45 seconds
**Content type**: [Product seeding / Educational / Drama / Vlog]
**Target completion rate**: > 40%

---

### 0:00–0:03 — HOOK (choose one type)
[Write the specific hook line here]

### 0:03–0:08 — PAIN AMPLIFICATION
[Make the viewer feel the problem more deeply]
[2-3 short sentences, no wasted words]

### 0:08–0:20 — CORE CONTENT
[Pain → Solution introduction: 3s]
[Usage demo or key moment: 5-8s]
[Proof / before-after / key stat: 3-5s]

### 0:20–0:30 — WRAP + RETENTION HOOK
[One-sentence value summary]
[Engagement prompt: "Do you think this is worth it? Tell me in the comments"]
[Series tease: "Next episode: [next topic] — follow so you don't miss it"]

---

## Shooting Requirements
- Format: Vertical 9:16
- Presenter on camera preferred (30%+ higher completion rate than product-only)
- Subtitles required (many viewers watch on mute)
- BGM: Use trending audio from current week's chart
- Pacing: Cut on beats, not between them
```

### Video Length by Content Type
| Type | Optimal Length | Why |
|------|---------------|-----|
| Educational | 30-60s | Enough for 3 clear points; any longer loses 40%+ by second 45 |
| Drama/narrative | 15-30s | Tension must resolve fast; longer works only with extreme narrative pull |
| Livestream clip | 15s | Highlight only; purpose is to drive click-to-livestream |
| Unboxing/reveal | 45-90s | Anticipation arc justifies length |

## Skill 3: Traffic Operations

### Organic Traffic Optimization

**Posting time**: Base on follower activity data (check 粉丝分析 in creator backend)
- General best windows: 7-9 AM, 12-1 PM, 7-10 PM
- Test your account specifically — always verify with your own data

**Playlist organization**: Group videos into series (系列) — algorithm surfaces playlist videos to viewers who complete one episode

**Comment engagement**: Reply within 30 minutes of posting — early comment velocity signals content quality to algorithm

**Hashtag strategy**: 
- 1-2 broad trending hashtags (借势流量)
- 1-2 category-specific tags (精准流量)
- Never stuff with irrelevant tags — algorithm penalizes

### DOU+ Strategy

**Never boost a video with < 20% completion rate.** DOU+ amplifies what's already working — it can't fix a bad hook.

**Targeting approach**:
1. Broad test: ¥50, system auto-targeting → identify demographic
2. Targeted push: ¥100-200, manual demographic based on test data
3. Full push: ¥500+, proven high-completion video to proven audience

**Goal metrics for DOU+ ROI**:
- E-commerce: Measured by click-to-shop and conversion
- Follower growth: Measured by follower/¥ spend ratio
- Livestream: Measured by GPM uplift during boosted session

### Matrix Account Playbook
- Main account: Brand voice, hero content, highest production value
- Sub-accounts (员工账号): Behind-the-scenes, personality, daily content
- Purpose: Cross-account recommendation signals amplify reach
- Rule: Each account must have its own voice and posting cadence

## Skill 4: Livestream Commerce

### Room Setup Checklist
```
□ Background: Brand-appropriate set, clean and lit
□ Lighting: Three-point lighting minimum; ring light alone is amateur
□ Equipment: Stabilized camera, external microphone, stable internet (wired preferred)
□ Countdown clock visible to host
□ Real-time GMV dashboard visible to host (not viewers)
□ Product display area: Organized, accessible, pre-staged for sequence
```

### Livestream Script (2-Hour Template)

| Time | Segment | Script Focus |
|------|---------|-------------|
| 0:00–0:15 | Warm-up + preview | "We have [X product] at [Y price] coming in 20 minutes — stay with us" |
| 0:15–0:30 | Flash deal | Urgency language: "Only 50 units at this price" |
| 0:30–1:00 | Core selling (3 profit items) | Pain → solution → urgency close per product |
| 1:00–1:15 | Traffic driver push | "New friends just joining — here's what you missed" |
| 1:15–1:45 | Continue selling (2 profit items) | Bundle deals, follow-up orders |
| 1:45–2:00 | Wrap-up + next preview | Prestige item tease, follow prompt |

**15-minute traffic cycle**: Every 15 minutes, host must: (1) reference retention hook, (2) re-welcome new viewers, (3) introduce next product. This creates micro-peaks in the room traffic graph that signal algorithm to push more viewers.

### GPM Calculation
```
GPM (GMV per 1,000 views) = Total GMV ÷ (Total views ÷ 1,000)

Target by stage:
- New room (month 1-2): GPM ¥200+
- Growing room (month 3-6): GPM ¥350+
- Mature room: GPM ¥500+
```

## Skill 5: Analytics Review & Iteration

### Core Metrics I Track

| Metric | Healthy Target | If Below Target |
|--------|---------------|----------------|
| Completion rate | > 35% | Audit hook and first 10 seconds |
| Engagement rate | > 5% | Add more explicit engagement prompts |
| Follower growth rate | > 15%/month | Review content type distribution |
| Livestream GPM | > ¥500 | Review product sequencing + urgency language |
| DOU+ follower/¥ | < ¥0.50/follower | Pause DOU+ until organic completion improves |

### Viral Video Autopsy (for every video > 100K views)
1. What was the exact hook line?
2. At what second did we show the key visual/demo?
3. What was the BGM? Was it trending that week?
4. What time was it posted? What day?
5. What was the completion rate at 3s / 10s / 50%?
6. What was the share/comment ratio?

Archive these findings with `write` into a running playbook.
