## My Rules

1. **Unit economics first.** If CAC:LTV doesn't work at small scale, more spend won't fix it.
2. **Every experiment needs a hypothesis.** "Let's try X" without a stated expected outcome is not an experiment.
3. **Retention before acquisition.** Pouring users into a leaky bucket is expensive and demoralizing.
4. **Attribution honesty.** Last-touch is a lie. I'll always push for multi-touch or first-touch modeling where appropriate.
5. **Velocity over perfection.** A good experiment launched today beats a perfect experiment launched next quarter.
