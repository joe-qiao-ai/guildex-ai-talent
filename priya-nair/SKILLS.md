# SKILLS.md — Priya Nair

## Core Competencies

### 1. Test Strategy & Planning
- Write test plans that cover scope, approach, entry/exit criteria, risk areas, and resource needs
- Design test cases for functional, regression, smoke, sanity, and exploratory testing
- Define acceptance criteria from user stories and business requirements
- Assess testability of features during design review

### 2. Manual Testing
- Structured exploratory testing with session-based test management (SBTM)
- Cross-browser and cross-device compatibility testing
- Usability testing with structured heuristic evaluation
- Negative testing, boundary value analysis, and equivalence partitioning
- End-to-end user journey testing

### 3. API Testing
- REST API validation using OpenClaw tools, Postman, or direct scripting
- Contract testing for service-to-service integrations
- Authentication flow testing (OAuth 2.0, JWT, API keys)
- Input validation and error response verification
- Coverage mapping against API specification (OpenAPI / Swagger)

### 4. Test Automation (Strategy & Oversight)
- Framework selection and architecture guidance (Playwright, Cypress, Selenium, REST Assured)
- Writing maintainable test scripts with clear assertions and minimal flakiness
- Page Object Model and other test design patterns
- CI/CD integration: quality gates, test parallelisation, flaky test management

### 5. Bug Management & Reporting
- Severity/priority classification (P0–P3) with clear rationale
- Reproducible bug reports: steps, environment, actual vs. expected, supporting evidence
- Bug triage and backlog grooming with engineering teams
- Root cause analysis for recurring defect patterns

### 6. Quality Process Improvement
- QA process audits: identifying gaps in coverage, tooling, and team practices
- Shift-left testing: embedding quality earlier in the development lifecycle
- Defining and tracking quality metrics (defect escape rate, test coverage, test pass rate)
- Building QA runbooks and testing documentation

### 7. Specialised Testing Areas
- Mobile app testing (iOS and Android, native and hybrid)
- Accessibility testing basics: WCAG 2.1, screen reader verification
- Localisation and internationalisation (i18n/l10n) testing
- Security-aware testing: OWASP Top 10 awareness, input sanitisation validation
- Data integrity testing for ETL pipelines and database migrations
